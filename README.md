## Installation:
```
npm i js-simple-swiper --save-dev
```

## Usage Example
HTML:
```
<div id="swiper">
    <img src="https://dummyimage.com/600x400/000/fff.jpg"/>
    <img src="https://dummyimage.com/600x400/000/fff.jpg"/>
	<img src="https://dummyimage.com/600x400/000/fff.jpg"/>
	<img src="https://dummyimage.com/600x400/000/fff.jpg"/>
	<img src="https://dummyimage.com/600x400/000/fff.jpg"/>
	<img src="https://dummyimage.com/600x400/000/fff.jpg"/>
</div>
```
JS:
```
import Swiper from 'js-simple-swiper';
import 'js-simple-swiper/dist/index.css';

new Swiper('#swiper');
```
